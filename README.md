# 日本株AI診断ツール

日本株の銘柄コードを入力するだけで、PER・PBR・ROE・配当利回り・自己資本比率を分析し、100点満点で「買い候補 / 様子見 / 危険」を自動判定するツールです。

## データソース

Yahoo Finance（非公式API）からリアルタイムデータを取得しています。APIキー不要。

| 指標 | 取得方法 |
|------|----------|
| 株価 | 実データ |
| PER | 実データ |
| PBR | 実データ |
| ROE | 実データ |
| 配当利回り | 実データ |
| 時価総額 | 実データ |
| 売上高 | 実データ |
| 営業利益（EBITDA） | 実データ |
| 自己資本比率 | 計算値（貸借対照表より） |

## スコアリング

| 指標 | 配点 |
|------|------|
| PER | 25点 |
| PBR | 15点 |
| ROE | 25点 |
| 配当利回り | 20点 |
| 自己資本比率 | 15点 |
| 合計 | 100点 |

- 65点以上：買い候補
- 45〜64点：様子見
- 44点以下：危険

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 を開いてください。

## デプロイ（Vercel）

Vercel に接続してそのままデプロイできます。環境変数は不要です。
